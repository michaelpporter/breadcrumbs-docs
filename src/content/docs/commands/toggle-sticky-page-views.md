---
title: "Toggle Sticky Page Views"
description: Command that toggles whether Breadcrumbs page views stay pinned in the note.
---
The **Toggle sticky page views** command flips the `sticky` setting for [page views](/views/page/) on or off.

When sticky is on, page views stay pinned in place as you scroll. Toggling redraws all open page views and shows a notice with the new state.
