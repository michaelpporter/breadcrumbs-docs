---
title: "Contributing"
description: Ways to contribute to Breadcrumbs, including filing issues, opening pull requests, and writing guides.
---
If you'd like to contribute to Breadcrumbs, you can do so in a few ways:

- [Add issues](https://github.com/michaelpporter/breadcrumbs/issues) for any bugs you find, or any useful features you'd like implemented.
- Open a PR on [the plugin repo](https://github.com/michaelpporter/breadcrumbs) to add a feature or fix an issue, or on [the docs repo](https://github.com/michaelpporter/breadcrumbs-docs) to improve the learning experience.
- Tell someone about Breadcrumbs, and how it works for your use-case. Writing a [Guide](/guides/) would be extremely valuable.
