---
title: "Metadata Field Suggester"
---
Breadcrumbs uses various metadata fields. See [Explicit Edge Builders](/explicit-edge-builders/explicit-edge-builders/), or [Note Attributes](/note-attributes/), for example.

The fields are automatically loaded as [Obsidian Properties](https://help.obsidian.md/Editing+and+formatting/Properties), so you get auto-completion when using them in Properties.
