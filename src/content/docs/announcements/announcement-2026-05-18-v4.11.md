## Breadcrumbs 🍞

**4.11.0** is out — one new feature for Mermaid codeblock views.

**New Features**

- **Per-field Mermaid arrow style** — each edge field now has an arrow-shape dropdown in Settings → Edge Fields. Set `==>` on your `same` field to get thick bidirectional arrows for sibling links while hierarchy links keep the default style. Edges sharing the same custom arrow on both directions collapse into a single bidirectional line (e.g. `==>` → `<==>`). ([#694](https://github.com/SkepticMystic/breadcrumbs/pull/694))
