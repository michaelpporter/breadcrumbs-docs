## Breadcrumbs Docs 🍞

📚 Major documentation update now live:

**New Guides** — six new step-by-step guides covering real-world use-cases:

- [Getting Started with Hierarchies](/guides/getting-started-with-hierarchies/) — the missing on-ramp for new users: edge fields, your first typed link, Matrix View and Trail View payoff, and common first-time mistakes
- [Zettelkasten Concept Hierarchy](/guides/zettelkasten-concept-hierarchy/) — atomic notes with Tag Notes for zero-maintenance MOCs, live codeblock trees, and Create List Index for static snapshots
- [Project and Task Management](/guides/project-and-task-management/) — four-level area/project/task hierarchy using List Notes and Folder Notes, with a transitive rule to trace any task back to its area
- [Academic Reading and Literature Notes](/guides/academic-reading-and-literature-notes/) — literature notes with Dataview Notes edge builder, implied authorship rules, auto-generated bibliographies, and Freeze Crumbs to File for publishing
- [Course-and-Curriculum-Builder](/guides/course-and-curriculum-builder/) — curriculum hierarchy via Dendron Notes, ordered lesson sequences with List Notes, collapsible codeblock trees, and Previous-Next View navigation
- [Vault Maintenance and Graph Auditing](/guides/vault-maintenance-and-graph-auditing/) — Graph Stats, `show-attributes` audits, `dataview-from` scoping, Freeze Crumbs to File, and CSS styling for implied vs. explicit edges

**Updated Docs**

- [Concepts](/concepts/) — added Edge Field Groups, Transitive Implied Relations, Implied Relation Rounds, Show Node Options, Lock View, Find Root, and Crumb Destination concept entries; fixed missing sort fields (`basename_natural`, `path_natural`) in Edge Sorters
- [Date Notes](/explicit-edge-builders/date-notes/) — added full Period Notes section documenting week/month/quarter/year hierarchy edges, per-period settings (Date Format, Folder, Next Field, Up Field), and a Mermaid diagram of the full temporal chain
- [Tree View](/views/tree-view/) — added Find Root and Find Root Field Groups settings; fixed Lock View and Lock Path descriptions
- [Matrix View](/views/matrix-view/) — added Custom Field Sorting, Field Order, Lock View, Lock Path, and Note Display Options
- [Trail View](/views/trail-view/) — added Merge Fields and Note Display Options; corrected Default Depth default value
- [Previous-Next View](/views/previous-next-view/) — split Field Groups into separate left (prevs) and right (nexts) pickers; added Period Rows section for Date Notes integration; added Note Display Options
- [Codeblocks](/views/codeblocks/) — added `markmap` type, `content` parameter (collapsible), and `field-prefix` parameter; added Note Display Options
