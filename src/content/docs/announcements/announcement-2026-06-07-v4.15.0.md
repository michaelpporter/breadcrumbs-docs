## Breadcrumbs 🍞

**4.15.0** released today — Breadcrumbs now targets Obsidian's new settings system, and the settings tab has been reorganised to be easier to navigate.

**Requires Obsidian 1.13**

This release requires Obsidian 1.13 or later. If you're still on Obsidian 1.12, stay on 4.14.2 — it's the last release that supports 1.12, and fixes for that line will be maintained separately.

**Searchable, navigable settings**

The settings tab now uses Obsidian 1.13's declarative settings API. Every section appears in the global settings search, and each area opens as its own page, so you can jump straight to what you need. The Breadcrumbs tab also has its own icon in the settings list.

**Reorganised settings**

- **Views** now lead with the in-note **Page** views (Trail, Previous/next) before the sidebar Matrix and Tree views.
- **Edge sources** are grouped by how they work: content you write (Tag, List), then names and paths (Dendron, Johnny.Decimal, Date, Regex), then link traversal (Traverse).
- Every settings entry now carries a short description explaining what it does.
